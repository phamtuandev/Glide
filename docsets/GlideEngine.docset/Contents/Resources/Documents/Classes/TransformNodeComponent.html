TransformNodeComponent

public final class TransformNodeComponent : GKSKNodeComponent, GlideComponent
extension TransformNodeComponent: DebuggableComponent

Component that defines the position and direction properties of an entity via rendering a container node on the scene for its entity’s other nodes.
